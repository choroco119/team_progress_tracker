# Team Progress Tracker

製番（製造番号）単位での工程進捗を管理するためのWebアプリケーションです。
社内LAN内の共有フォルダを活用し、サーバーレスで多人数による同時編集とデータ共有を実現します。

## 主な機能

- **多人数同時編集（共有フォルダ同期）**: 
  - ブラウザのFile System Access APIを使用し、サーバー不要で共有フォルダ上のデータを直接同期。
  - **排他制御（ロック機能）**: 同時書き込みによるデータ破損を防止。
  - **アクション時自動同期**: 登録・保存・削除などの操作時に即座に共有フォルダへ反映。
  - **自動バックグラウンド更新**: 30秒ごとの自動リフレッシュにより他人の変更を自動検知。
- **自動デイリーバックアップ**: 
  - 共有フォルダ内に `backups` フォルダを自動作成。
  - 毎日最初の接続時に自動でスナップショットを保存（直近30日分を保持）。
- **製番・工程管理**:
  - 納入仕様書、板金手配、部品手配、銘板手配、社内工事図、ソフト開発の進捗管理。
  - **インクリメンタル検索による案件コピー**: 新規登録時、過去の製番・客先・件名から検索して基本情報と工程要否を即座にコピー。
  - **インテリジェント・ツールチップ**: マウスオーバーで詳細情報を即座に表示。画面端での切れ込み防止機能を搭載。
- **高度なフィルタリング**: 
  - 工程別の「未完了案件」抽出や、自由度の高い検索機能。
- **マスターデータ管理**: 
  - 客先、規格、板金業者、担当者リストのカスタマイズ。

## 使い方

1. `index.html` をブラウザで開きます。
2. 画面右上の**「同期設定」**をクリックし、社内共有フォルダ内の保存先を選択します。
3. 案件の追加（新規登録モーダル内の検索窓から過去案件の情報をコピー可能）や、各工程セルをクリックして進捗を入力します。
4. 保存や登録ボタンを押すと、**自動的に共有フォルダへ同期**されます。
5. 他人の編集を即座に反映させたい場合は、右上の**「最新データに更新」**をクリックします。
6. 一覧表のセルを直接クリックすることで、工程詳細の入力や案件情報の編集が可能です。
7. ツールチップ（マウスオーバー）で各工程の詳細情報を即座に確認できます。
