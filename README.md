# Team Progress Tracker

製番（製造番号）単位での工程進捗を管理するためのWebアプリケーションです。
エンジニアリング、板金手配、部品手配、ソフト開発などの各工程の状態をリアルタイムで可視化します。

## 主な機能

- **製番管理**: 製番、納期、客先、件名、担当者などの基本情報を管理。
- **工程進捗の可視化**:
  - 納入仕様書（承認日）
  - 板金手配（見積、注文、納期確定）
  - 部品手配（主部品、予備品、支給品それぞれの納期管理）
  - 銘板手配
  - 社内工事図
  - ソフトウェア開発
- **高度なフィルタリング**: 製番、客先、担当者、品名などによる検索に加え、各工程の「未完了案件」のみを抽出可能。
- **マスターデータ管理**: 客先リスト、規格リスト、板金業者リスト、担当者リストを個別に管理。
- **データ永続化**: JSONファイルとしての読込・保存に対応（ブラウザのファイルAPIを使用）。
- **レスポンシブデザイン**: ダークモードを採用したモダンで視認性の高いUI。

## 使い方

1. `index.html` をブラウザで開きます。
2. 右上の「新規製番登録」から案件を追加します。
3. 一覧の各工程セルをクリックすると、詳細な進捗（日付やメモ）を入力できるモーダルが開きます。
4. 画面上部の「保存」ボタンをクリックして、現在の状態をJSONファイルとして保存できます。
5. 次回利用時は「読込」ボタンから保存したJSONファイルを読み込んでください。
