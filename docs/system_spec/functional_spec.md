# 機能仕様書 (Functional Specification)

本アプリケーションが提供する主要な機能の詳細について記述します。

## 1. プロジェクト（製番）管理
製番単位で案件を登録し、その進捗を一目で確認できる一覧表を提供します。

- **新規登録**: 基本情報（製番、客先、件名、納期、担当者等）を入力。
- **編集**: 一覧表のセル（製番情報エリア）をクリックして基本情報を修正。
- **削除**: 案件を完全に削除（確認ダイアログあり）。

## 2. 工程進捗管理
以下の6つの工程を個別に管理できます。

1. **納入仕様書**: 出図、返却期日、返却、修正要否、承認を管理。
2. **板金手配**: 見積依頼、外注先、注文、図面出図、リミット、検収を管理。
3. **部品手配**: 「本体」「予備」「支給」の3カテゴリに分けて手配依頼・納期を管理。
4. **銘板手配**: 手配依頼、納期を管理。
5. **社内工事図**: 出図、メモを管理。
6. **ソフト開発**: 作成、デバッグを管理。

## 3. インクリメンタル検索による案件コピー
新規登録モーダル上部で過去の製番を検索し、情報を即座にコピーできます。

- **コピー対象**: 客先、件名、向先、品名、数量、規格、担当者、検査、および**各工程の要・不要（necessity）**設定。
- **目的**: 類似案件の入力工数を最小限に抑える。

## 4. 高度なフィルタリング・ソート
大量の案件から目的の情報を素早く見つけ出すための機能です。

- **キーワードフィルタ**: 製番、客先、件名、向先、品名、担当者で絞り込み。
- **詳細工程フィルタ（セレクトボックス）**: 
  - **納入仕様書**: 未出図、未返却、修正要、未承認で絞り込み。
  - **板金手配**: 見積依頼未送付、注文書未送付、詳細図未送付で絞り込み。
  - **部品手配**: 未手配（要設定がすべて済でない）で絞り込み。
  - **銘板手配**: 未手配（注文書送付が未）で絞り込み。
  - **社内工事図**: 未出図（出図が未）で絞り込み。
  - **ソフト**: 未作成、未デバッグで絞り込み。
- **完了案件の表示制御**: 
  - デフォルトでは完了済み案件は非表示。
  - 「完了を表示」ボタンで表示の有無を切り替え可能。
  - **重要**: いずれかの工程フィルタを使用している間は、完了済み案件は強制的に除外されます。
- **ソート**: 製番、納期での昇順・降順並べ替え。

## 5. インテリジェント・ツールチップ
一覧表の各行にマウスを合わせることで、詳細情報を即座に確認できます。

- **表示内容**: 各工程の日付データ、メモ、予算情報など。
- **画面端制御**: ツールチップがブラウザの端で切れないよう、表示位置を自動調整します。

## 6. マスターデータ管理
「設定」モーダルから、以下の共有リストを編集できます。
- 客先名（読みカナ対応）
- 規格リスト
- 板金外注先リスト
- 担当者リスト
