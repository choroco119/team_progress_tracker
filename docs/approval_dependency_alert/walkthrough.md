# 変更内容のまとめ (Walkthrough) - 承認連携アラート

納入仕様書の「承認」完了に伴い、着手可能となる銘板手配と板金の詳細図送付に対するアラート機能を実装しました。

## 1. 銘板手配のアラート
- **条件**: 納入仕様書が「承認済」かつ、銘板手配が「未着手（注文書送付前）」の場合。
- **表示**: `！要着手` アイコンを表示します。

## 2. 板金手配（詳細図送付）のアラート
- **条件**: 納入仕様書が「承認済」かつ、板金手配が「注文書送付済」だが「詳細図が未送付」の場合。
- **表示**: `！詳細図送付可` アイコンを表示します。注文書送付済みの情報のすぐ上に表示されるため、次のステップが明確になります。

## 3. 検証内容
- 納入仕様書を承認済みにすると、銘板手配に「要着手」、板金手配に「詳細図送付可」のアラートが表示されることを確認。
- 銘板の注文書を送付すると、銘板のアラートが消えることを確認。
- 板金の詳細図を送付すると、板金のアラートが消えることを確認。
