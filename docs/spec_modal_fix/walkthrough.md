# 納入仕様書モーダル修正 完了報告

納入仕様書モーダルにおける「修正要否」ボタンの選択状態が正しく表示されない問題を修正しました。

## 修正内容

### クラス管理の排他化
CSSにおいて `btn-outline` が `btn-primary` よりも後に定義されているため、両方のクラスが付与されていると `btn-outline`（背景透明）が優先されてしまうことが原因でした。
そのため、`btn-primary` を付与する際には `btn-outline` および `btn-toggle` を削除し、逆に `btn-primary` を削除する際には `btn-outline` を付与するようにロジックを修正しました。

### 修正箇所
- **renderSpecDocModal**: 「修正要否（要・否）」ボタンの状態反映ロジックを修正。
- **setBtnState**: 各工程モーダル内で「済」ボタン等の状態を管理する共通パーツを修正（計6箇所）。
