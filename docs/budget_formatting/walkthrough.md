# 変更内容の解説（板金予算のカンマ区切り対応）

板金予算の表示を「100,000」のようなカンマ区切り形式に対応させました。

## 主な変更点

### 1. 入力時の自動フォーマット
`app.js` の `setupInputFormatters` を更新し、板金予算の入力欄に数字を打ち込むとリアルタイムでカンマが挿入されるようにしました。これにより、入力ミスを防ぎ、金額の桁数を把握しやすくなっています。

### 2. ツールチップ表示の改善
一覧画面で各行にマウスを合わせた際に表示される詳細情報のツールチップにおいて、予算額がカンマ区切りで表示されるように修正しました。

### 3. 編集・コピー機能の整合性
既存の製番を編集する場合や、他の案件から情報をコピーした場合でも、予算額が正しくカンマ区切りでフォームにセットされるように調整しました。

## 技術的な詳細
- `app.js` に `formatNumberWithCommas` 関数を追加し、一貫したフォーマット処理を行っています。
- `Number.prototype.toLocaleString()` を利用して、ブラウザの標準的な地域設定に基づいたカンマ区切りを実現しています。

- [x] ツールチップで「100,000」のように表示されること
