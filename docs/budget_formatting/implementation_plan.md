# 板金予算のカンマ区切り表示の実装計画

板金予算（金額）の視認性を向上させるため、数値を100,000のようにカンマ区切りで表示・入力できるようにします。

## 変更内容

### 1. ヘルパー関数の追加 (app.js)
- `formatNumberWithCommas(val)`: 数値をカンマ区切りの文字列に変換する共通関数を実装します。

### 2. UI表示の修正 (app.js)
- テーブルの行にマウスホバーした際に表示されるツールチップ内の「板金予算」をカンマ区切りにします。

### 3. 入力フォームの修正 (app.js, index.html)
- 入力時にリアルタイムでカンマを挿入するバリデーション/フォーマッタを実装します。
- モーダルを開いた際や「既存案件からコピー」した際にもカンマ区切りで初期表示されるようにします。

## 検証計画
- 新規登録時に数値を入力し、リアルタイムでカンマが挿入されることを確認。
- 登録後、ツールチップでカンマ区切りで表示されることを確認。
- 編集モーダルを開いた際に、保存された値がカンマ区切りで表示されることを確認。
