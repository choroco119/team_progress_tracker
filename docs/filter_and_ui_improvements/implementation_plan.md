# 実装計画書 (Implementation Plan)

## 概要
Engineering Progress Tracker の利便性向上のため、各工程の詳細フィルタ実装およびUIの改善を行う。

## 実施項目
1.  **詳細工程フィルタの拡充**: 板金、部品、銘板、社内図、ソフトの各工程に詳細な絞り込み機能を追加する。
2.  **完了案件の表示制御**: デフォルト非表示化と、「完了を表示」ボタンによる切り替えを実装する。
3.  **UIハイライトの改善**: マウスホバー時に製番情報と操作箇所を連動してハイライトさせる。
4.  **ドキュメント整理**: 古いドキュメントを整理し、最新仕様書を作成する。

## 技術的アプローチ
- `index.html` のフィルタエリアをセレクトボックス形式に統一。
- `app.js` の `renderTable` 内のフィルタリングロジックを AND 条件で拡張。
- CSS の `:has` セレクタおよびホバー設定の調整。
