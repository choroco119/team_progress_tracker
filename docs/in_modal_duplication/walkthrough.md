# モーダル内コピー機能への移行完了 (Walkthrough)

一覧表のレイアウトを元のスッキリした状態に戻し、コピー機能を新規登録モーダル内へ統合しました。

## 変更内容

### 1. 一覧表UIの復旧
- 一覧表から「操作」列を削除しました。これにより、以前と同様の広々としたレイアウトでデータを確認いただけます。

### 2. モーダル内コピー機能
- **「新規製番登録」モーダル**を開いた際、最上部に「既存案件から情報をコピー」というセクションが表示されるようになりました。
- ドロップダウンからコピーしたい過去の製番を選択して「適用」ボタンを押すと、その案件の基本情報と工程の要否設定が現在のフォームに自動入力されます。

### 3. 表示の最適化
- このコピー機能は**新規登録時のみ**表示されます。既存案件の「編集」時には表示されないため、誤操作を防ぎ、迷わずに入力いただけます。

## 検証済み項目
- [x] 一覧表から「操作」列が消えていること。
- [x] 新規登録モーダルを開くと、既存案件のリストが正しく表示されること。
- [x] 既存案件を選択して「適用」を押すと、客先や工程の要否などが正しく反映されること。
- [x] 既存案件の編集（セルをクリックして開く）時には、コピー用UIが表示されないこと。
