# リファクタリング：ユーティリティ関数の抽出 完了レポート

## 概要
`app.js` から汎用的なヘルパー関数を `js/utils.js` に分離しました。これにより、本体のコードが約100行削減され、ロジックの整理が進みました。

## 変更点

### 1. 新規ファイル：[utils.js](js/utils.js)
- 以下の関数をグローバルスコープで定義し、どこからでも利用可能にしました。
    - 数値/日付フォーマット関連
    - 期限判定ロジック
    - UIヘルパー（`showToast`）
    - ストレージヘルパー（`openDB`）

### 2. メインロジック：[app.js](app.js)
- 重複する関数定義を削除しました。
- `openDB` の呼び出しを、新しい汎用的なインターフェース（`openDB(DB_NAME, STORE_NAME)`）に合わせて修正しました。

### 3. エントリポイント：[index.html](index.html)
- スクリプトの読み込み順序を調整し、依存関係を解決しました。

## 結果
- コードの可読性が向上しました。
- 既存の機能（予算のフォーマット、トースト表示、同期設定の保存など）への影響がないことをコードレビューで確認しました。
