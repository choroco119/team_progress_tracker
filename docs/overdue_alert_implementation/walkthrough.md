# 変更内容のまとめ (Walkthrough) - 納入仕様書期日アラート

納入仕様書の返却期日が超過している案件を視認しやすくするため、アラート機能（赤字表示）を実装しました。

## 1. 期日超過の判定ロジック
新しいヘルパー関数 `isOverdue` を導入し、以下の条件で判定を行っています。
- `spec.dueDate`（返却期日）が設定されている。
- `spec.returnDate`（返却日）が未入力である。
- `spec.dueDate` が今日または過去の日付である。

## 2. UIの変更
一覧表の「納入仕様書」カラムにおいて、上記の条件を満たす場合に**返却期日を赤字かつ太字**で表示するようにしました。

| ステータス | 表示内容の例 |
| :--- | :--- |
| **期限内** | 出図済<br><small>返却期日:2026/05/10</small> |
| **期限超過 (今日を含む)** | 出図済<br><small style="color:var(--danger-color);font-weight:bold;">返却期日:2026/05/02</small> |

## 3. 検証内容
- 期限超過の案件が期待通り赤字で表示されることを確認。
- 返却日を入力（返却済の状態に）すると、赤字表示が消えることを確認。
- 期日が未設定の案件でエラーが発生しないことを確認。
