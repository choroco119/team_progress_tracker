# 納入仕様書の返却期日超過アラート（赤字表示）

納入仕様書の返却期日が今日または過去の日付であり、かつ返却が済んでいない場合に、一覧表上で期日を赤字で表示するようにします。

## ユーザーレビューが必要な項目
- なし

## 変更内容

### [JS] [app.js](file:///c:/Users/kohei/.gemini/antigravity/scratch/team_progress_tracker/app.js)

1.  **ヘルパー関数の追加**: `isOverdue(dateStr)` を追加し、指定された日付が今日または過去であるかを判定できるようにします。
2.  **テーブルレンダリングの修正**: `renderTable` 関数内の納入仕様書（`specDoc`）のレンダリング箇所を修正します。
    - `issueDate` があり、`returnDate` がなく、かつ `dueDate` が超過している場合に `var(--danger-color)` を適用します。

## 検証計画
### 手動検証
- 納入仕様書の返却期日を「今日」または「過去の日付」に設定し、返却日が未入力の状態にする。
- 一覧表の「返却期日」が赤字（太字）で表示されることを確認する。
- 返却日を入力すると、赤字表示が消えることを確認する。
