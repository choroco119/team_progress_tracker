# クイック同期機能の実装完了

一度選択した同期フォルダをブラウザに記憶させ、次回以降は簡単な操作（または自動）で同期を開始できる「クイック同期」機能を実装しました。

## 実装内容

### 1. UIの更新
- ヘッダーに **「クイック同期」** ボタンを追加しました。
- このボタンは、過去にフォルダを選択したことがある場合にのみ表示されます。
- デザインはプライマリーカラーの枠線と背景（薄い青）で視認性を高めています。

### 2. IndexedDB による永続化
- ブラウザの IndexedDB を使用して、`FileSystemDirectoryHandle` を保存する仕組みを導入しました。
- これにより、ブラウザを閉じたりリロードしたりしても、選択したフォルダの情報を保持できます。

### 3. 同期フローの改善
- **自動再接続**: ページ読み込み時に保存されたフォルダがあり、かつ権限が既に付与されている場合は自動的に同期を開始します。
- **ワンクリック接続**: 権限が必要な場合は「クイック同期」ボタンが表示されます。これをクリックしてブラウザの許可ダイアログで「OK」を押すだけで同期が始まります。
