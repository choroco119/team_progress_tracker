# 変更内容のまとめ (Walkthrough) - 社内工事図 連携アラート

社内工事図の着手に必要な前工程が全て揃ったことを通知するアラート機能を実装しました。

## 1. アラートの発生条件
以下の前工程が全て「完了」または「不要」であり、かつ社内工事図が「未着手（出図前）」の場合に表示されます。
- **板金手配**: 詳細図送付済（または不要）
- **部品手配**: 全ての必要な部品カテゴリ（主・予・支）が手配依頼済
- **銘板手配**: 注文書送付済（または不要）

## 2. UIの表示
社内工事図のセルに、オレンジ色のアラートアイコン（！）と「要着手」というテキストを表示します。

| ステータス | 表示例 |
| :--- | :--- |
| **前工程待ち** | - |
| **着手可能（全前工程完了）** | <i data-lucide="alert-circle" style="color:var(--warning-color);"></i> **要着手** |

## 3. 検証内容
- 板金・部品・銘板の全てを完了状態にすると、社内工事図に「要着手」が表示されることを確認。
- いずれか一工程でも未完了（例：銘板がまだ）の場合は、アラートが表示されないことを確認。
- 社内工事図を出図済みにすると、アラートが消えることを確認。
