# 製番コピー登録機能の実装完了 (Walkthrough)

既存の製番情報をベースに、新しい製番として効率的に登録できるコピー機能を実装しました。

## 実装内容

### 1. 「操作」列とコピーボタンの追加
- 一覧表の「納期」の隣に「操作」列を新設し、コピーアイコン（`copy`）を配置しました。
- 各行のコピーアイコンをクリックすることで、その案件の情報を引き継いだ状態で新規登録画面が開きます。

### 2. 賢い情報の引き継ぎ
- **引き継ぐ情報**: 客先、件名、納期、担当者、名称、数量、リンク、備考、および **各工程の要否設定（例：ソフト要/不要など）**。
- **引き継がない（リセットする）情報**: 
    - **製番（ID）**: 新しく入力していただくため空欄になります。
    - **進捗状況**: 日付やステータス、メモなどの進捗データはすべてクリアされ、未着手の状態で作成されます。

### 3. モードの自動切り替え
- コピーボタンから開いた際はタイトルが「製番コピー登録」になり、保存時は「更新」ではなく「新規登録」として処理されます。
- 登録完了後は通常通り、共有フォルダへの自動同期が行われます。

## 検証済み項目
- [x] コピーボタンクリックで、製番以外が正しく入力されたモーダルが開くこと。
- [x] 「要否」の設定がソース元と同じになっていること。
- [x] 登録後、新しい製番が一覧に追加され、元の案件の進捗（日付等）は引き継がれていないこと。

---
これにより、類似案件の登録作業が大幅にスピードアップします。
