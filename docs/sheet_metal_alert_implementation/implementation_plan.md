# 板金手配の詳細図送付期日の超過アラート

板金手配において、詳細図送付期日が今日または過去の日付であり、かつ詳細図送付が完了していない場合に、期日を赤字で表示するようにします。

## ユーザーレビューが必要な項目
- なし

## 変更内容

### [JS] [app.js](file:///c:/Users/kohei/.gemini/antigravity/scratch/team_progress_tracker/app.js)

1.  **テーブルレンダリングの修正**: `renderTable` 関数内の板金手配（`sheetMetal`）のレンダリング箇所を修正します。
    - `poSentDate`（注文書送付済）のステータス表示において、`drawingDueDate` が設定されており、かつ `isOverdue` である場合に `var(--danger-color)` を適用します。

## 検証計画
### 手動検証
- 板金手配で「注文書送付済」にし、詳細図送付期日を「今日」または「過去の日付」に設定する（詳細図送付日は未入力）。
- 一覧表の「詳細図送付期日」が赤字（太字）で表示されることを確認する。
- 詳細図送付日を入力すると、赤字表示が消える（またはステータスが「詳細図送付済」に変わる）ことを確認する。
