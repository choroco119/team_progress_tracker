# 変更内容のまとめ (Walkthrough) - 板金手配 詳細図送付期日アラート

板金手配における詳細図送付期日の超過を視認しやすくするため、アラート機能（赤字表示）を実装しました。

## 1. 期日超過の判定ロジック
納入仕様書と同様に、以下の条件で判定を行っています。
- `sm.drawingDueDate`（詳細図送付期日）が設定されている。
- `sm.drawingSentDate`（詳細図送付日）が未入力である。
- `sm.drawingDueDate` が今日または過去の日付である。

## 2. UIの変更
一覧表の「板金手配」カラムにおいて、ステータスが「注文書送付済」の場合、**詳細図送付期日を赤字かつ太字**で表示するようにしました。

| ステータス | 表示内容の例 |
| :--- | :--- |
| **期限内** | 注文書送付済<br><small>納期(注文書):2026/05/10<br>詳細図送付期日:2026/05/05</small> |
| **期限超過 (今日を含む)** | 注文書送付済<br><small>納期(注文書):2026/05/10<br><span style="color:var(--danger-color);font-weight:bold;">詳細図送付期日:2026/05/02</span></small> |

## 3. 検証内容
- 詳細図送付期日が超過している案件で、期待通り赤字・太字で表示されることを確認。
- 詳細図を送付（詳細図送付日を入力）すると、ステータスが「詳細図送付済」に変わり、赤字表示が消えることを確認。
